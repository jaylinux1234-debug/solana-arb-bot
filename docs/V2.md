# Focused v2 bot (dex-cheap reverse only)

Minimal replacement path for the full multi-strategy monitor. One lane, one pair, three gates.

## What v2 does

- **Pair:** SOL/USDC only
- **Lane:** `dex_cex_reverse` when Jupiter is cheaper than Backpack bid (`dex_cheap`)
- **Gates:** `V2_MIN_GROSS_BPS`, `V2_MIN_NET_BPS`, live Jupiter re-quote (`ROUNDTRIP_PRE_SIM_OK`)
- **Log:** `logs/v2_attempts.jsonl` only (no near-miss file)
- **Does not:** forward CEX-DEX, brain router, near-miss spam, webhooks, collateral, liquidation

## Run locally

```bash
# Terminal 1
npm run ledger:bridge

# Terminal 2 — uses your existing .env
npm run v2:start
```

Health (default port **8001** to avoid clashing with v1 monitor on 8000):

```bash
curl http://127.0.0.1:8001/health
```

## Run in Docker (alongside or instead of v1)

Stop v1 monitor first (same Ledger / RPC), then:

```bash
npm run sync:compose-env
docker compose run --rm -e BOT_HEALTH_PORT=8001 solana-arb-monitor python -m src.v2.main
```

Or add a dedicated compose service later; for now `python -m src.v2.main` in the monitor image is enough.

## Env reference

| Variable | Default | Meaning |
|----------|---------|---------|
| `V2_MIN_GROSS_BPS_BASE` | 7.0 | Base gross gate (scaled by vol) |
| `V2_MIN_NET_BPS_BASE` | 1.2 | Base net gate (scaled by vol) |
| `V2_ADAPTIVE_VOL_ENABLED` | true | Scale gates with `VolatilityTracker` |
| `V2_VOL_LOOKBACK_MIN` | 5 | Rolling bid window (minutes) |
| `V2_MAX_FLASH_USDC` | 12 | Max trade size (USDC) |
| `V2_POLL_INTERVAL_SEC` | 2.8 | Seconds between cycles |
| `V2_BASE_COST_BPS` / `V2_SLIPPAGE_BUFFER_BPS` | 7.5 / 2.8 | Cost model drag |
| `V2_USE_REALISTIC_COST_MODEL` | true | Use v2 cost stack vs legacy |
| `V2_ADAPTIVE_THRESHOLDS` | true | Lower gross min when 5m vol high |
| `V2_HEALTH_PORT` | 8001 | Health server port |
| `V2_SINGLETON_KEY` | `bot:singleton:v2` | Redis lock (separate from v1) |
| `BACKPACK_API_KEY` / `BACKPACK_SECRET` | — | Required for CEX bid (or `*_FILE` in `secrets/`) |

## Compare attempts

```bash
python -c "
import json
from collections import Counter
c = Counter()
for line in open('logs/v2_attempts.jsonl'):
    o = json.loads(line)
    c[o.get('block_reason') or o.get('execution_status') or '?'] += 1
print(c.most_common(10))
"
```
