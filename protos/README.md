# Geyser / Yellowstone gRPC protos

Place `geyser.proto` (and any imports) here, then build the image or run:

```bash
sh scripts/compile_geyser_protos.sh
```

Upstream sources: [rpcpool/yellowstone-grpc](https://github.com/rpcpool/yellowstone-grpc).

Generated Python modules land in `src/core/generated/`.
